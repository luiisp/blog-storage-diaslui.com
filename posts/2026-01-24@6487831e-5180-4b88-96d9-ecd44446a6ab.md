## O que é o neovim?
  Se voce programa provavelmente voce deve saber o que é o VS Code ou outro editor de texto para código, mas em resumo, é um programa como o seu bloco de notas que voce usa para guardar suas anotações
  mas como objetivo de editar arquivos de código.
  
  E olha só, o Vs code deixou de ser um editor de código (code editor) e se tornou um editor de codigo com IA (ai code editor)
  
  <img width="741" height="137" alt="image" src="https://github.com/user-attachments/assets/dad03d62-d221-453e-8459-1b7428f36a79" />
  
  > Nem o VS Code escapou da era da IA

  Não dá fugir da IA, é realmente bom, mas, isso é papo pra outra hora.

  A questão é que com o avanço da IA parece que tem IA em tudo, seu windows tem ia, seu editor de código, sua pesquisa. As vezes, simplesmente voce quer codar, puramente, seja em C, C++ ou qualquer coisa low-level,
  a minha sugestão de voce usar esse tal do neovim vem disso

  **Se voce precisa:**
   - Codar puramente (se IA)
   - Codar rapidamente sem perder tempo
   - Codar num pc ruim ou sem pesar tanto o pc

  Talvez, voce precise do neovim.

  O neovim é um editor de código como o Vs Code, mas com uma filosofia bem peculiar, a filosofia Vim.
  Basicamente, a filosofia vim é a filosofia de você escrever código rapidamente e SEM O MOUSE
  Sim, tudo que voce faria em outro editor de código clicando em botoes e arrastando com o mouse, voce faz diretamente apertando teclas do teclado.
